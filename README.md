# Subhajit Portfolio

A production-grade engineering portfolio built with Next.js, TypeScript, Tailwind CSS, shadcn/ui, and MDX.

The goal of this project is to showcase engineering thinking, backend architecture, system design, and product ownership rather than just frontend development.

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- MDX
- next-themes
- Lucide Icons
- Vercel

## Engineering Principles

- Scalability over shortcuts
- Simplicity over cleverness
- Performance by default
- Accessibility first
- Strong typing
- Clean architecture
- Reusable components
